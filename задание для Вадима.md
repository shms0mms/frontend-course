
# Практическое задание по React Hook Form

## 🎯 Тема: Конструктор плана обучения

### 📌 Задача

Создать форму «Создание учебного плана», где пользователь может составить собственный план изучения технологии.

---

## 📝 Поля формы

### 1️⃣ Основная информация

- Название плана (обязательное, минимум 5 символов)
    
- Описание (textarea, максимум 300 символов)
    
- Уровень сложности (radio: Beginner / Intermediate / Advanced)
    
- Предполагаемая длительность (число недель, от 1 до 52)
    

---

### 2️⃣ Список модулей (динамический)

Пользователь может добавлять модули обучения.

Каждый модуль содержит:

- Название модуля (обязательное)
    
- Количество часов (минимум 1)
    
- Тип материала (select: видео / статья / практика)
    
- Чекбокс «Обязательный модуль»
    

Реализовать можно через `useFieldArray`.

---

### 3️⃣ Навыки, которые будут получены

- Массив строк (input + кнопка "Добавить")
    
- Минимум 2 навыка
    
- Без дубликатов
    

---

## ⚙️ Требования

### ✅ Валидация

- Использовать `register`
    
- Проверять:
    
    - минимум 1 модуль
        
    - минимум 2 навыка
        
    - суммарное количество часов всех модулей ≤ 200
        
- Отображать ошибки под соответствующими полями
    

---

### ✅ Поведение формы

- Кнопка Submit неактивна, если форма невалидна
    
- После отправки:
    
    - вывести структурированные данные в консоль
        
    - очистить форму через `reset`
        
- Показать общее количество часов в реальном времени (использовать `watch`)
