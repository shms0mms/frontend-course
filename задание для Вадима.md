## 3. ResponsiveNavbar.jsx (адаптивная задача)

Задача:

- На desktop: меню горизонтальное
    
- На mobile: бургер-кнопка
    
- Меню открывается вертикально
    
- Использовать `useState`
    
- Tailwind breakpoints (`md`, `lg:`)
    

```jsx
import React, { useState } from "react";

const ResponsiveNavbar = () => {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <nav>
      <div>
        <h1>MyLogo</h1>

        <button onClick={() => setIsOpen(!isOpen)}>
          ☰
        </button>

        <ul>
          <li>Home</li>
          <li>About</li>
          <li>Services</li>
          <li>Contact</li>
        </ul>
      </div>
    </nav>
  );
};

export default ResponsiveNavbar;
```
