
### 1. Содержимое текстовых узлов и комментариев

Создайте HTML:

```html
<div id="box">Привет</div>
```

Задача:

1. Получите текст внутри `div`.
    
2. Добавьте в `div` комментарий `<!-- комментарий -->`.
    
3. Измените текст внутри `div` на `"Привет, мир!"`.
    

---

### 2. Создание новых элементов и узлов

Задача:

1. Создайте новый элемент `<li>`.
    
2. Добавьте в него текст `"Новый пункт"`.
    
3. Добавьте этот элемент в список:
    

```html
<ul id="list"></ul>
```

---

### 3. Методы вставки

Дан HTML:

```html
<div id="container">
  <p>Первый</p>
</div>
```

Задача:

1. Создайте новый `<p>` с текстом `"Второй"`.
    
2. Вставьте его **в конец `container`**.
    
3. Вставьте ещё один `<p>` **в начало контейнера**.
    

---

### 4. Метод `insertAdjacentHTML`

Есть HTML:

```html
<div id="box"></div>
```

Задача:

1. С помощью `insertAdjacentHTML` добавьте:
    
    - `<p>До элемента</p>` перед `box`
        
    - `<p>После элемента</p>` после `box`
        
    - `<span>Внутри</span>` в конец `box`.
        

---

### 5. `insertAdjacentText` и `insertAdjacentElement`

Есть:

```html
<div id="block"></div>
```

Задача:

1. Добавьте текст `"Hello"` **в начало блока** через `insertAdjacentText`.
    
2. Создайте `<b>World</b>` и добавьте его **в конец блока** через `insertAdjacentElement`.
    

---

### 6. Перенос элемента

Дан HTML:

```html
<div id="box1">
  <p id="text">Текст</p>
</div>

<div id="box2"></div>
```

Задача:  
Переместите `<p id="text">` из `box1` в `box2`.

---

### 7. Клонирование узлов (`cloneNode`)

Есть:

```html
<div class="card">
  <h3>Заголовок</h3>
  <p>Описание</p>
</div>
```

Задача:

1. Клонируйте `.card` полностью.
    
2. Добавьте копию в конец `body`.
    
3. Измените текст заголовка у клона.
    

---

### 8. Удаление узлов

Дан список:

```html
<ul id="list">
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>
```

Задача:

1. Удалите второй элемент списка.
    
2. Очистите весь список.
    

---

### 9. Управление классами (`classList`)

Есть элемент:

```html
<div id="box" class="red"></div>
```

Задача:

1. Добавьте класс `active`.
    
2. Удалите класс `red`.
    
3. Переключите класс `hidden`.
    
4. Проверьте, есть ли класс `active`.
    

---

### 10. Работа со стилями и атрибутами

HTML:

```html
<div id="square" data-size="100"></div>
```

Задача:

1. Получите значение `data-size`.
    
2. Установите ширину и высоту `square` равными этому значению (в `px`).
    
3. Установите цвет фона через `style`.
    
4. Получите **вычисленный стиль** ширины через `getComputedStyle`.
    
5. Перезапишите все стили через `cssText`.
