

Тебе нужно расположить три иконки (или просто текста) внутри одного контейнера так, чтобы они разлетелись по разным сторонам: одна — **слева**, вторая — **ровно по центру**, третья — **справа**.

#### Что нужно сделать:

1. Создай один родительский блок (`div`).
    
2. Внутри него создай три дочерних элемента (например, три `span`).
    
3. С помощью Flexbox:
    
    - Поставь их в одну линию.
        
    - Распредели их так, чтобы первый прилип к левому краю, последний — к правому, а средний остался в центре.
        
    - Сделай так, чтобы они не прилипали к верхнему краю контейнера, а стояли по его центру (вертикально).
        

---

### Твой шаблон

**HTML:**

```
<div class="toolbar">
  <div class="item">Назад</div>
  <div class="item">Играет трек: Let it be</div>
  <div class="item">Меню</div>
</div>
```

**CSS:**

```
.toolbar {
  display: flex; /* Делаем контейнер гибким */
  
  /* 1. Как распределить элементы по всей ширине? */
  /* Твой код здесь */
  
  /* 2. Как выровнять их по вертикали по центру? */
  /* Твой код здесь */

  height: 60px;
  background-color: #333;
  color: white;
  padding: 0 20px;
}
```

### Подсказка:

Тебе понадобятся всего две основные строки в CSS:

- `justify-content` с нужным значением
    
- `align-items` для вертикали.
