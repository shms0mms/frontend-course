

## 1. Что такое Shadcn UI

**Shadcn UI** — это набор готовых компонентов интерфейса для React, построенный на основе:

- React
    
- Tailwind CSS
    
- Radix UI
    
- TypeScript
    

Главная особенность — **это не классическая библиотека компонентов**.  
Компоненты **копируются в ваш проект**, и вы можете изменять их код.

Преимущества:

- полный контроль над компонентами
    
- удобная кастомизация
    
- современный дизайн
    
- высокая доступность (accessibility)
    
- отлично подходит для React и Next.js
    

Официальный сайт: [https://ui.shadcn.com](https://ui.shadcn.com/)

---

# 2. Технологии внутри Shadcn UI

Shadcn UI использует несколько технологий.

## Tailwind CSS

Используется для стилизации компонентов.

```html
<button class="bg-black text-white px-4 py-2 rounded">
  Click
</button>
```

---

## Radix UI

Radix UI — это **headless компоненты**.

Они предоставляют:

- логику
    
- доступность
    
- управление состоянием
    

Но **не имеют стилей**.

Примеры компонентов Radix:

- Dialog
    
- Tooltip
    
- Tabs
    
- Dropdown Menu
    
- Popover
    

---

## Class Variance Authority (CVA)

Используется для создания **вариантов компонентов**.

Например у кнопки могут быть варианты:

- primary
    
- outline
    
- destructive
    

---

# 3. Установка Shadcn UI

## 1. Создание проекта

Чаще всего используется Next.js.

```bash
npx create-next-app@latest my-app
```

---

## 2. Установка Tailwind

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

---

## 3. Инициализация Shadcn

```bash
npx shadcn-ui@latest init
```

После этого появится папка:

```
components/ui
```

---

# 4. Добавление компонентов

Компоненты добавляются через CLI.

Например кнопка:

```bash
npx shadcn-ui add button
```

Создается файл:

```
components/ui/button.tsx
```

---

# 5. Использование компонента Button

```tsx
import { Button } from "@/components/ui/button"

export default function Page() {
  return (
    <Button>Click me</Button>
  )
}
```

---

## Варианты кнопки

```tsx
<Button>Default</Button>

<Button variant="outline">
  Outline
</Button>

<Button variant="destructive">
  Delete
</Button>
```

---

## Размеры

```tsx
<Button size="sm">Small</Button>

<Button size="lg">Large</Button>
```

# 7. Работа с формами

Shadcn UI отлично работает вместе с **React Hook Form**.

Пример поля формы:

```tsx
<FormField
  control={form.control}
  name="email"
  render={({ field }) => (
    <FormItem>
      <FormLabel>Email</FormLabel>

      <FormControl>
        <Input {...field} />
      </FormControl>

      <FormMessage />
    </FormItem>
  )}
/>
```

---

# 8. Темная тема

Shadcn поддерживает **Dark Mode**.

Пример:

```html
<html class="dark">
```

Tailwind классы:

```
bg-white dark:bg-black
```

# 9. Кастомизация компонентов

Компоненты можно редактировать прямо в проекте.

Например:

```

```

Можно менять:

- стили
    
- размеры
    
- поведение
    
- варианты
