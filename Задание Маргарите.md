
Мини-сайт: **«Расписание тренировок»**. На компьютере это карточки в один ряд, на телефоне — одна под другой.

задача: адаптировать под мобильное устройство

### 1. HTML (`index.html`)

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Мое Расписание</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <header>
        <h1>Расписание на завтра</h1>
    </header>

    <div class="schedule-container">
        <div class="card">
            <div class="time">08:00</div>
            <h3>Йога</h3>
            <p>Растяжка и дыхание</p>
        </div>

        <div class="card active">
            <div class="time">12:00</div>
            <h3>Кроссфит</h3>
            <p>Интенсивная работа</p>
        </div>

        <div class="card">
            <div class="time">19:00</div>
            <h3>Бокс</h3>

</body>
</html>
```

### 2. CSS (`style.css`)


```css
body {
    font-family: sans-serif;
    margin: 0;
    padding: 20px;
    background-color: #f0f2f5;
}

header {
    text-align: center;
    margin-bottom: 30px;
}

/* Контейнер для карточек */
.schedule-container {
    display: flex;         /* Включаем Flexbox */
    gap: 20px;             /* Расстояние между карточками */
    justify-content: center; /* Центрируем карточки */
}

/* Стили карточки */
.card {
    flex: 1;               /* Карточки делят место поровну */
    background: white;
    padding: 20px;
    border-radius: 12px;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
    text-align: center;
}

.active {
    border: 2px solid #ff4757;
    background: #fffafa;
}

.time {
    font-size: 24px;
    font-weight: bold;
    color: #ff4757;
}
```

### Твое задание:

1. **Поменяй порядок:** Сделай так, чтобы на мобильной версии (внутри `@media`) карточки шли в обратном порядке (самая нижняя стала верхней).
        
2. **Добавь отступы:** Сейчас на мобилке карточки будут прилипать к краям экрана. Добавь в `@media` для `.schedule-container` небольшой `padding`.
    
3. **Бонус:** Попробуй изменить цвет заголовка `h1` только для мобильной версии.
