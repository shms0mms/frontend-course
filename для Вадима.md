
### Твоя задача:

Реализуй компонент `UserList`, который должен выполнять следующие условия:

1. **useRef (Фокус):** При первом рендере страницы (или по нажатию на специальную кнопку «Focus Search») фокус должен автоматически устанавливаться в поле ввода (`input`).
    
2. **useMemo (Фильтрация):** У тебя есть массив из 5000+ пользователей (создай его искусственно). При вводе текста в поиск, список должен фильтроваться. Используй `useMemo`, чтобы тяжелая фильтрация не запускалась при каждом обновлении интерфейса (например, если в компоненте тикают часы или меняется другая тема оформления).
    
3. **useRef (Сохранение стейта без рендера):** Реализуй счетчик того, сколько раз пользователь нажал на список, но **не** отображай это число в UI сразу. Выведи это значение в консоль только при размонтировании компонента или по специальной кнопке «Show Stats».

### Стартовый шаблон (заготовка):


```
import React, { useState, useMemo, useRef, useEffect } from 'react';

// Имитация большого списка данных
const createUserList = () => {
  return Array.from({ length: 5000 }, (_, i) => ({
    id: i,
    name: `User ${i}`,
    email: `user${i}@example.com`,
  }));
};

const UserList = () => {
  const [query, setQuery] = useState('');
  const [theme, setTheme] = useState('light'); // Для проверки лишних рендеров
  const users = useMemo(() => createUserList(), []);

  // --- ТВОЙ КОД ТУТ ---
  // 1. Создай ref для input
  // 2. Создай ref для хранения количества кликов (clickCount)
  // 3. Используй useMemo для фильтрации массива users на основе query
  // --------------------

  return (
    <div style={{ background: theme === 'light' ? '#fff' : '#333', color: theme === 'light' ? '#000' : '#fff' }}>
      <button onClick={() => setTheme(t => t === 'light' ? 'dark' : 'light')}>
        Сменить тему (не должно тормозить фильтрацию)
      </button>
      
      <hr />
      
      <input 
        // Привяжи ref
        value={query}
        onChange={(e) => setQuery(e.target.value)}
        placeholder="Search users..."
      />
      
      <button>Focus Search</button>

      <ul>
        {/* Рендери здесь отфильтрованный список */}
      </ul>
    </div>
  );
};

export default UserList;
```
