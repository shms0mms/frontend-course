
## Задание: "Theme Switcher"

Тебе нужно создать простое приложение, где заголовок и кнопка меняют свой цвет в зависимости от выбранной темы, хранящейся в контексте

### Шаги выполнения:

1. **Создай контекст:** Назови его `ThemeContext`. По умолчанию тема может быть `"light"`.
    
2. **Создай Провайдер:** Оберни всё приложение (или корневой компонент) в `ThemeContext.Provider`.
    
3. **Добавь состояние:** Используй `useState`, чтобы хранить текущую тему (`light` или `dark`) и функцию для её переключения. Передай их в `value` провайдера.
    
4. **Создай компоненты:**
    
    - `Header`: Просто выводит текст "Текущая тема: [название]".
        
    - `Button`: При клике меняет тему в контексте.
        
5. **Потреби контент:** Внутри `Header` и `Button` используй хук `useContext(ThemeContext)`, чтобы получить данные.
    

### Ожидаемое поведение:

При нажатии на кнопку в компоненте `Button`, текст в компоненте `Header` должен мгновенно обновляться, хотя ты **не передаешь** никаких пропсов напрямую от родителя к детям.

---

### Маленькая подсказка по структуре кода:


```jsx
// 1. Создание
const ThemeContext = createContext();

// 2. Обертка
<ThemeContext.Provider value={{ theme, toggleTheme }}>
  <AppContent />
</ThemeContext.Provider>

// 3. Использование
const { theme } = useContext(ThemeContext);
```
